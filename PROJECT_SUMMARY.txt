================================================================================
                            HARDWARE WIRING
================================================================================

GPIO CONNECTIONS:
  Pan Servo Signal   → GPIO 17 (PWM)
  Tilt Servo Signal  → GPIO 27 (PWM)
  Servo Power        → 5V
  Servo Ground       → GND

RECOMMENDED:
  - 5V/2A+ power supply for servos
  - Proper servo brackets/gimbal mount
  - RPi camera module on gimbal tip
  - Shielded signal wires

================================================================================
                          HARDWARE INTEGRATION
================================================================================

CURRENT STATE: Simulation mode (PWM values logged, not applied to GPIO)

TO ENABLE REAL GPIO CONTROL:
  1. Install pigpio:
     $ sudo apt-get install pigpio
     $ sudo systemctl enable pigpiod
     $ sudo systemctl start pigpiod

  2. Edit CMakeLists.txt
     Uncomment the pigpio section (lines 47-52)

  3. Rebuild:
     $ cmake .. && make

  4. Run with GPIO access:
     $ sudo ./bin/gimbal_example
